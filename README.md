### Datumize Challenge

[UI Component Documentation](https://dist-ihuaclnbji.now.sh/src-components-add-item-form-index#basic-usage)

To run it locally

Install the packages:

```sh
yarn
```

Start the local db:

```sh
yarn start:db
```

Start the server:

```sh
yarn start
```
