### Datumize Challenge

[UI Component Documentation](https://dist-jooukvzitv.now.sh/)

To run it locally

Install the packages:

```sh
yarn
```

Start the local db:

```sh
yarn start:db
```

Start the server:

```sh
yarn start
```
